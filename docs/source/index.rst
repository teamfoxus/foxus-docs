Foxus documentation
===================================

[Foxus](http://www.foxus.com/) is a project experimenting with passthrough Augmented Reality on existing devices. It uses a stereoscopic camera add-on that runs on the Oculus Quest 2 at 60fps, with full RGB passthrough. 

The Foxus is built with community support by the team at [voxels.com](https://www.voxels.com/). 

.. note::

   This project is under active development.
